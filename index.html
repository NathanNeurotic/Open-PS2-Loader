RiptOPL Documentation

An opinionated fork of Open PS2 Loader aiming to be the "definitive build" — a cover-art Coverflow menu by default, a Favourites tab, per-game Neutrino external-core launching, a consolidated Device Settings hub, PS1 support via POPSTARTER, and every storage backend the PS2 can reach.

Open PS2 Loader (OPL) is a 100% open-source game/app loader for the PS2 and backward-compatible PS3. RiptOPL is a downstream fork with opinionated defaults and extra features layered on top. Its settings live in their own settings_riptopl.cfg so it never collides with stock OPL or wOPL on the same memory card — while artwork, themes, VMCs and favourites stay shared. The Getting Started guide walks you through your first game; the sections below are the deep reference. For the canonical project, see ps2homebrew/Open-PS2-Loader.

What RiptOPL adds on top of OPL

  • Coverflow theme (default) — a centered cover-art carousel with reflection, animated scrolling, a configurable cover count, and aspect-correct covers in 4:3 and widescreen. Tunable live under Coverflow Settings.
  • Favourites tab — press R3 to star any game; a virtual Favourites page gathers starred games from every device. Imports an existing uOPL / wOPL favourites.bin.
  • Neutrino external core (per-game) — hand a game to an external neutrino.elf instead of OPL's built-in core, with a Device picker and a structured Launch Args screen.
  • Network boot — stream games from a PC over the LAN as their own list via UDPBD or UDPFS (Neutrino).
  • PS1 games via POPSTARTER — a per-device L3 "VCD" view lists your PS1 *.VCD games, including from an exFAT drive (BDMAssault) and the internal HDD.
  • Device Settings hub — one page for per-device options, cache sizes, Block-Device (BDM) settings, all MMCE settings, network-boot controls, and the Favourites toggle.
  • Ready-to-use defaults — widescreen, cover art, notifications, sound + boot sound, USB and the PS2 logo are on out of the box; DualSense (PS5 controller, USB) is an optional build.

What's the difference between OPL, RiptOPL, and wOPL?

Open PS2 Loader (OPL) is the canonical, community-maintained loader at ps2homebrew/Open-PS2-Loader. wOPL (KrahJohlito) and uOPL (Wolf3s) are forks where much of the modern Coverflow / Favourites / Neutrino work originated. RiptOPL is a downstream "agglomeration" fork that brings those features together with opinionated defaults and additional device support, while staying compatible with stock OPL on the same memory card (separate settings_riptopl.cfg, shared art/themes/VMC/favourites). It is a labor of love, not a replacement for upstream — please support the canonical project.
